# Joyspoon
JoySpoon is a frontend-based Mukhvas ordering web application designed for academic demonstration. The project simulates an online food ordering system where users can browse food items, add them to a cart, and place orders.
This project was developed as part of a Full Stack Development (FSD) college project to demonstrate frontend development concepts and user interface design.
Features
### 1.Product Browsing
Users can explore different mukhwas products, combos, and gift hampers available on the website.
2.Add to Cart System
Users can add products to the shopping cart and view the total items and price
3.Cart Management
View cart items
Remove items from cart
Clear entire cart
View total price
4.Product Detail View
5.Responsive Design
The website is fully responsive and mobile-friendly using Bootstrap.
6.Product Categories
Products are organized into different categories such as:
Bestselling Mukhwas
Premium Mukhwas
Combo Packs
Gift Hampers
Deal Corner
7.Contact & Social Integration
Users can contact the brand and connect through:
Instagram
YouTube
WhatsApp chat button
